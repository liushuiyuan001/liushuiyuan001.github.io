# 性能优化

1. 减少垃圾回收

   ```
   不用的对象及时置null
   清空数组的时候不但要赋值给空数组，同时将数组的长度赋值为0
   函数的优化：在循环或者函数内部的函数可以提取到外面
   ```

2. 减少内存泄漏

   ```
   意外的全局变量
   被遗忘的计时器或回调函数
   脱离DOM的引用
   闭包
   ```
